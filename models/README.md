# Vosk Models

To enable offline transcription, you need to download a Vosk model and extract it here.

Recommended for French:
[vosk-model-small-fr-0.22](https://alphacephei.com/vosk/models/vosk-model-small-fr-0.22.zip) (~45 MB)

**Steps:**
1. Download the zip file.
2. Extract it into this folder.
3. Rename the extracted folder to `vosk-model-fr`.

The directory structure should look like:
`models/vosk-model-fr/am/`
`models/vosk-model-fr/conf/`
...
